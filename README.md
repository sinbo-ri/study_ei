# 📚 塾 社会科対策サイト

次男の塾の教材をベースにした一問一答形式の学習サイトです。

## 📖 概要

- **対象範囲**: 
  - 第11回: 富山県・新潟県（20問）
  - 第12回: 山形県・福島県（20問）
  - 全問題: 40問
- **形式**: 一問一答（記述式）
- **出題モード**: 順番通り / ランダム

## ✨ 機能

### 基本機能
- ✅ 一問一答形式での出題
- ✅ 正解・不正解の即座判定
- ✅ 授業回ごとの問題選択（第11回・第12回・全問題）
- ✅ 連続正解のメッセージ表示（5問、10問、15問、20問）
- ✅ 進捗状況の表示（問題数、正解数、連続正解数）
- ✅ 順番通り / ランダム出題の切り替え
- ✅ スマホ・タブレット対応のレスポンシブデザイン

### 連続正解メッセージ
- 5問連続正解: 🎉 5問連続正解！！ 🎉
- 10問連続正解: 🔥 10問連続正解！！ 🔥
- 15問連続正解: ⭐ 15問連続正解！！ ⭐
- 20問連続正解: 👑 全問正解！！完璧です！ 👑

## 🚀 使い方

### ローカルで動作確認

1. すべてのファイルを同じフォルダに配置
2. `index.html` をブラウザで開く

### 問題の追加・編集

`questions.csv` を編集します：

```csv
問題,答え,回
富山県の県庁所在地はどこですか？,富山,11
山形県の県庁所在地はどこですか？,山形,12
```

- **問題**: 出題する問題文
- **答え**: 正解（大文字小文字、全角半角は自動で判定）
- **回**: 授業回（11 または 12）

### GitHubにアップロード

詳細は `DEPLOY_GUIDE.md` を参照してください。

## 🎯 学習のコツ

1. **まずは順番通りで**: 基本問題を覚えましょう
2. **ランダムで復習**: 知識が定着しているか確認
3. **苦手な回を集中**: 第11回・第12回を選んで重点学習
4. **連続正解を目指す**: モチベーション維持に効果的

## 📱 スマホでの使い方
