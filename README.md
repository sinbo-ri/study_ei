# 📚 塾 対策クイズサイト

小学生向けの理科・社会の復習用クイズサイトです。

## ✨ 機能

- **科目選択**: 理科・社会・全科目から選択
- **授業回選択**: 第1回〜第20回、または全授業回から選択
- **四択問題形式**: 選択肢はランダムに並び替え
- **出題モード**: 順番通り / ランダム
- **解説表示**: 回答後に詳しい解説を表示
- **連続正解ボーナス**: 5問、10問、15問、20問連続で特別メッセージ
- **進捗表示**: 現在の進捗、正解数、連続正解数を表示
- **完了画面**: 全問終了後に正答率を表示
- **レスポンシブデザイン**: スマホ・タブレット対応

## 🎮 使い方

1. **科目を選択**: 社会のみ、理科のみ、または全科目
2. **授業回を選択**: 第1回〜第20回、または全授業回
3. **出題モード選択**: 
   - 「順番通り」: CSVの順番通りに出題
   - 「ランダム」: ランダムな順序で出題
4. **問題に回答**: 4つの選択肢から正しいものをクリック
5. **解説を確認**: 回答後に表示される解説で理解を深める
6. **次の問題へ**: ボタンをクリックして次の問題へ進む

## 📝 問題の追加・編集方法

### CSVファイルの形式

`questions.csv` をExcelやテキストエディタで編集できます。

**フォーマット:**
```
問題,選択肢1,選択肢2,選択肢3,選択肢4,正解,解説,科目,回
```

**例:**
```
富山県の県庁所在地はどこですか？,富山,金沢,新潟,長野,1,富山県の県庁所在地は富山市です。,社会,11
```

### 各列の説明

- **問題**: 問題文
- **選択肢1〜4**: 4つの選択肢
- **正解**: 正解の番号（1〜4）
- **解説**: 回答後に表示される解説文
- **科目**: `社会` または `理科`
- **回**: `1` から `20` までの数字

### 問題を追加する手順

1. `questions.csv` をExcelまたはGoogleスプレッドシートで開く
2. 最後の行に新しい問題を追加
3. 上記のフォーマットに従って入力
4. ファイルを保存
5. ブラウザで強制リロード（Ctrl + Shift + R）

## 🎨 カスタマイズ

### 色の変更

`style.css` の以下の部分を編集：

```css
:root {
    --primary-color: #43CEA2;    /* メインカラー */
    --secondary-color: #185a9d;  /* サブカラー */
    --correct-color: #4CAF50;    /* 正解の色 */
    --incorrect-color: #f44336;  /* 不正解の色 */
}
```

## 📱 スマホのホーム画面に追加

### iOS (Safari)
1. ページを開く
2. 共有ボタン（四角と矢印）をタップ
3. 「ホーム画面に追加」をタップ

### Android (Chrome)
1. ページを開く
2. メニュー（三点リーダー）をタップ
3. 「ホーム画面に追加」をタップ
